一款轻量级 macOS 虚拟显示器管理工具。创建虚拟 Retina 显示器、管理 HiDPI 分辨率，并能通过菜单栏GUI或CLI控制。

---

## 功能特性

### 核心功能 (v1.0)

- 创建自定义分辨率的虚拟显示器
- HiDPI/Retina 支持 (1920×1080、2560×1440、3840×2160)
- 多种刷新率可选 (30/60/120/144 Hz)
- 菜单栏集成，快速访问
- 通过弹出菜单创建/删除虚拟显示器
- 菜单栏显示显示器状态和数量

### 增强功能 (v1.1)

- 预设配置 (办公、阅读、演示、游戏)
- CLI 工具支持 (`m2d-cli start`、`m2d-cli stop`、`m2d-cli list`、`m2d-cli create` 等)

| GPU | 集成显卡 | 独立显卡（用于 120/144 Hz） |

### 所需权限

虚拟显示器需要特定的系统权限才能正常工作：

**屏幕录制权限（必需）**

- **位置**: 系统偏好设置 → 安全性与隐私 → 隐私 → 屏幕录制
- **用途**: 允许应用程序创建和管理虚拟显示器
- **提示时**: 在显示器创建对话框出现时授予权限
- **检测**: 应用程序自动检测缺失权限并显示"请求权限"按钮

**辅助功能权限（可选）**

- **位置**: 系统偏好设置 → 安全性与隐私 → 隐私 → 辅助功能
- **用途**: 启用自动排列等高级显示器管理功能
- **使用时机**: 仅在特殊显示器排列和快捷键功能时需要

**权限请求流程**:
1. 应用程序在启动或显示器创建时检测缺失的权限
2. 通过 UI 提示用户授予权限
3. 系统自动打开相关的偏好设置面板
4. 授予权限后继续显示器创建

## 使用说明

### 菜单栏

m2d 作为菜单栏应用程序运行。点击菜单栏中的 m2d 图标（📊）访问：

- **虚拟显示器数量和状态**: 显示当前/总显示器数和主刷新率
- **快捷操作**: 创建显示器、排列（系统偏好设置）、设置
- **所有显示器列表**: 带开关切换的物理和虚拟显示器
- **移除全部**: 一次性删除所有虚拟显示器

### 创建虚拟显示器

1. 点击菜单栏中的 m2d 图标
2. 点击"创建显示器"按钮（蓝色主按钮）
3. **步骤 1 - 选择分辨率**:
   - 选择预设分辨率（1920×1080、2560×1440、3840×2160）或点击"自定义"
   - 选择刷新率（30/60/120/144 Hz）
   - 查看 GPU 负载指示器
4. **步骤 2 - 确认**:
   - 验证配置摘要
   - 检查系统状态（绿色 = 就绪）
   - 点击"创建显示器"
5. 如果提示，在系统偏好设置中授予屏幕录制权限
6. 虚拟显示器将出现在显示器列表中


### CLI 命令

```bash
# 启动 m2d 后台应用程序
m2d-cli start

# 停止 m2d 并移除所有虚拟显示器
m2d-cli stop

# 列出所有虚拟显示器
m2d-cli list

# 创建虚拟显示器
m2d-cli create --width 1920 --height 1080 --refresh-rate 60 --hidpi

# 创建带自定义名称的显示器
m2d-cli create --width 2560 --height 1440 --refresh-rate 120 --name "我的虚拟显示器"

# 通过 ID 移除虚拟显示器
m2d-cli remove <显示器-id>

# 启用虚拟显示器
m2d-cli enable <显示器-id>

# 禁用虚拟显示器
m2d-cli disable <显示器-id>

# 收藏（保存）虚拟显示器
m2d-cli favorite <显示器-id>

# 取消收藏虚拟显示器
m2d-cli unfavorite <显示器-id>

# 显示帮助
m2d-cli --help
```

## 故障排除

### 虚拟显示器创建失败

**权限错误**:
- 在系统偏好设置 → 安全性与隐私 → 隐私 → 屏幕录制中授予屏幕录制权限
- 如果应用不在列表中，退出并重启应用，然后重试
- 使用创建对话框中的"请求权限"按钮进行自动设置

**"系统未注册显示器"错误**:
- 确保已授予屏幕录制权限
- 等待几秒后重试
- 检查系统报告 → 图形/显示器以验证显示器检测

**应用显示权限警告**:
- 点击创建对话框中的"请求权限"按钮
- 这将自动打开系统偏好设置
- 授予权限并在需要时重启应用

### 性能问题

**120/144 Hz 高 GPU 使用率**:
- 对于集成显卡，将刷新率降低到 60 Hz
- 减少虚拟显示器数量
- 关闭其他 GPU 密集型应用程序
